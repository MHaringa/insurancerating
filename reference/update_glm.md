# Deprecated alias for `refit_glm()`

`update_glm()` is deprecated as of version 0.8.0. Use
[`refit()`](https://mharinga.github.io/insurancerating/reference/refit.md)
for the new refinement workflow.

## Usage

``` r
update_glm(x, intercept_only = FALSE, ...)
```

## Arguments

- x:

  Object of class `rating_refinement`, `restricted` or `smooth`.

- intercept_only:

  Logical.

- ...:

  Other arguments.

## Value

See
[`refit_glm()`](https://mharinga.github.io/insurancerating/reference/refit_glm.md).
