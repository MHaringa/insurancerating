# Deprecated alias for `fisher_classify()`

`fisher()` is deprecated as of version 0.8.0.

## Usage

``` r
fisher(x, n = 7, diglab = 2)
```

## Arguments

- x:

  Numeric vector to classify.

- n:

  Number of classes.

- diglab:

  Deprecated. Use `dig.lab` in
  [`fisher_classify()`](https://mharinga.github.io/insurancerating/reference/fisher_classify.md)
  instead.

## Value

See
[`fisher_classify()`](https://mharinga.github.io/insurancerating/reference/fisher_classify.md).
