# Deprecated alias for `extract_model_data()`

`model_data()` is deprecated in favour of
[`extract_model_data()`](https://mharinga.github.io/insurancerating/reference/extract_model_data.md).

## Usage

``` r
model_data(x)
```

## Arguments

- x:

  An object of class `"glm"`, `"refitsmooth"`, or `"refitrestricted"`.

## Value

See
[`extract_model_data()`](https://mharinga.github.io/insurancerating/reference/extract_model_data.md).
