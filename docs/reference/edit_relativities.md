# Edit a relativities step

Edit a relativities step

## Usage

``` r
edit_relativities(
  ref,
  relativities,
  variable = NULL,
  step = NULL,
  normalize = NULL
)
```

## Arguments

- ref:

  Object of class `rating_refinement`.

- relativities:

  Named list of data.frames.

- variable:

  Optional variable name.

- step:

  Optional step index.

- normalize:

  Optional logical override.

## Value

Object of class `rating_refinement`.
