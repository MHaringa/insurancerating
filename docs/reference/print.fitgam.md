# Print method for fitgam objects

Prints the prediction component of a `fitgam` object created by
[`riskfactor_gam()`](https://mharinga.github.io/insurancerating/reference/riskfactor_gam.md).

## Usage

``` r
# S3 method for class 'fitgam'
print(x, ...)
```

## Arguments

- x:

  An object of class `"fitgam"`.

- ...:

  Further arguments passed to or from other methods (ignored).

## Value

Invisibly returns `x`.
