# Edit a restriction step

Edit a restriction step

## Usage

``` r
edit_restriction(ref, restrictions, variable = NULL, step = NULL)
```

## Arguments

- ref:

  Object of class `rating_refinement`.

- restrictions:

  New restriction data.frame.

- variable:

  Optional variable name.

- step:

  Optional step index.

## Value

Object of class `rating_refinement`.
