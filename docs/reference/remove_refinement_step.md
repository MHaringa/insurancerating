# Remove a refinement step

Remove a refinement step

## Usage

``` r
remove_refinement_step(ref, variable = NULL, step = NULL, type = NULL)
```

## Arguments

- ref:

  Object of class `rating_refinement`.

- variable:

  Optional variable name.

- step:

  Optional step index.

- type:

  Optional step type.

## Value

Object of class `rating_refinement`.
